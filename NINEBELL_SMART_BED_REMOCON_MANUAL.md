# NINEBELL SMART CARE BED 리모컨 사용 매뉴얼

---

## 1. 제품 개요

NINEBELL SMART CARE BED 리모컨은 스마트 침대의 모든 기능을 제어하는 전용 무선 리모컨입니다.

- **MCU**: adStar ae32000 EISC
- **디스플레이**: ILI9488 3.5인치 TFT LCD (320x480, RGB565)
- **통신**: ESP32 UART (115200bps)
- **키 입력**: SPI Shift Register (16비트)

---

## 2. 버튼 배치 및 기능

리모컨에는 총 15개의 물리 버튼이 있습니다.

| 버튼 | 기능 | 비트 위치 |
|------|------|-----------|
| **전원** (POWER) | 숏클릭: 초기위치 복귀, 롱클릭(~3초): 종료 | Bit 0 |
| **체압분산** (VAIRANCE) | 체압분산 모드 진입 | Bit 1 |
| **교대부양** (LEVITATE) | 교대부양 모드 진입 | Bit 2 |
| **마사지** (MASSAGE) | 마사지 모드 진입 | Bit 3 |
| **환자케어** (CARE) | 환자케어 모드 진입 | Bit 4 |
| **확인** (CONFORM) | 시작/정지 토글, Enable/Disable 선택 | Bit 5 |
| **▲** (UP) | 커서 위로 이동 | Bit 6 |
| **▼** (DOWN) | 커서 아래로 이동 / 배변 모드 평탄화 | Bit 8 |
| **◀** (LEFT) | 설정값 감소 / 모드 전환 / 배변 좌 틸팅 | Bit 9 |
| **▶** (RIGHT) | 설정값 증가 / 모드 전환 / 배변 우 틸팅 | Bit 10 |
| **설정/저장** (SET) | 설정 모드 진입 및 저장 | Bit 11 |
| **초기화** (INIT) | 초기화 모드 진입 | Bit 12 |
| **온열** (HEAT) | 온열 단계 전환 (OFF↔1↔2↔3, LED 표시) | Bit 13 |
| **통풍** (VENTIL) | 통풍 단계 전환 (OFF↔1↔2↔3, LED 표시) | Bit 14 |

> **참고**: 키 입력은 Active LOW 방식입니다 (버튼 누름 시 해당 비트가 0).

---

## 3. 전원 ON/OFF

- **전원 ON**: 전원 버튼을 1회 짧게 누릅니다. LCD가 켜지고 홈 화면이 표시됩니다.
- **전원 OFF (종료)**: 전원이 켜진 상태에서 전원 버튼을 **약 3초간 길게** 누릅니다. 3초가 되면 즉시 "종료합니다." 화면이 표시되며, 침대가 초기위치로 복귀한 후 LCD가 꺼집니다. 종료 진행 중에는 **모든 버튼이 차단**됩니다.
- **초기위치 복귀 (숏클릭)**: 전원이 켜진 상태에서 전원 버튼을 짧게 눌렀다 떼면 동작 정지 + 초기위치 복귀 + 홈 화면으로 전환됩니다.
- 전원이 꺼진 상태에서는 **전원 버튼 외의 모든 버튼이 무시**됩니다.

> **참고**: 롱클릭은 버튼을 떼는 시점이 아니라 **3초 도달 시점에 즉시** 종료가 시작됩니다.

---

## 4. 부팅 모드

전원 인가 시 GPIO 0.2, 0.3 핀 상태에 따라 3가지 모드로 부팅됩니다.

| GPIO 0.3 | GPIO 0.2 | 모드 |
|-----------|-----------|------|
| 1 | 1 | 일반 부팅 (Normal) |
| 1 | 0 | 매뉴얼/자가진단 모드 |
| 0 | 1 | Mass Storage 모드 (USB) |

---

## 5. 홈 화면 (HOME)

전원 ON 후 기본으로 표시되는 화면입니다. 홈 화면에서 각 기능 버튼을 눌러 해당 모드로 진입합니다.

---

## 6. 동작 모드별 사용법

### 6.1 체압분산 모드 (VAIRANCE)

체압분산 버튼을 눌러 진입합니다.

**화면 구성**:
- 상단: 타이틀 바 + 메인 아이콘
- 좌측: 신체부위 라디오 버튼 (머리/몸통(상)/몸통(하)/다리)
- 우측: 설정값 표시 (레벨, 속도)

> **참고**: 7x10 체압 히트맵 기능은 현재 비활성화 상태입니다.

**조작 방법**:

| 버튼 | 동작 |
|------|------|
| **체압분산** | 홈 화면으로 복귀 |
| **설정/저장** | 설정 모드 진입 → 저장 (토글) |
| **▲ / ▼** | 신체부위 선택 (머리↔몸통(상)↔몸통(하)↔다리) |
| **◀ / ▶** | 설정값 감소/증가 (레벨, 속도) |
| **확인** | 시작/일시정지/재시작 (일반 모드) / Enable/Disable 토글 (설정 모드) |

**설정 항목**:
- **레벨**: 1~6단계
- **속도**: 1~6단계

---

### 6.2 교대부양 모드 (LEVITATE)

교대부양 버튼을 눌러 진입합니다. 3가지 서브 모드(일반/집중/수면)를 지원합니다.

**화면 구성**:
- 상단: 타이틀 바 + 메인 아이콘
- 서브 모드 선택: 일반 | 집중 | 수면 (아이콘)
- 중앙: BAR 상태 표시 + 인체 이미지
- 좌측: 신체부위 라디오 버튼 (머리/몸통(상)/몸통(하)/다리)
- 우측: 설정값 표시 (시간설정, 높이, 속도)

**조작 방법**:

| 버튼 | 동작 |
|------|------|
| **교대부양** | 홈 화면으로 복귀 |
| **◀ / ▶** (설정 모드 아닐 때) | 서브 모드 전환 (일반↔집중↔수면) |
| **설정/저장** | 설정 모드 진입 → 저장 (토글) |
| **▲ / ▼** | 신체부위 선택 또는 설정값 항목 이동 |
| **◀ / ▶** (설정 모드일 때) | 설정값 감소/증가 |
| **확인** | 시작/일시정지/재시작 (일반 모드) / Enable/Disable 토글 (설정 모드) |

**설정 항목** (신체부위별):
- **시간설정**: 10분 / 20분 / 30분 / 1시간 / 6시간 / 12시간 / 24시간
- **높이**: 30 / 60 / 90 / 120 / 150 / 170
- **속도(RPM)**: 200 / 250 / 300

---

### 6.3 마사지 모드 (MASSAGE)

마사지 버튼을 눌러 진입합니다. 12가지 마사지 패턴을 지원합니다.

**화면 구성**:
- 상단: 타이틀 바 + 메인 아이콘
- 마사지 패턴 아이콘 (12종)
- 중앙: BAR 상태 표시 + 인체 이미지
- 좌측: 신체부위 라디오 버튼
- 우측: 설정값 표시 (시간설정, 높이)

**조작 방법**:

| 버튼 | 동작 |
|------|------|
| **마사지** | 홈 화면으로 복귀 |
| **◀ / ▶** (설정 모드 아닐 때) | 마사지 패턴 전환 (1~12) |
| **설정/저장** | 설정 모드 진입 → 저장 (토글) |
| **▲ / ▼** | 신체부위 선택 또는 설정값 항목 이동 |
| **◀ / ▶** (설정 모드일 때) | 설정값 감소/증가 |
| **확인** | 시작/일시정지/재시작 (일반 모드) / Enable/Disable 토글 (설정 모드) |

**설정 항목** (신체부위별):
- **시간설정**: 10분~120분 (10분 단위)
- **높이**: 30 / 60 / 90 / 120 / 150 / 170

---

### 6.4 환자케어 모드 (PATIENT CARE)

환자케어 버튼을 눌러 진입합니다. 2가지 기능(머리감기/배변)을 지원합니다.

**화면 구성**:
- 상단: 타이틀 바 + 메인 아이콘
- 서브 기능 선택: 머리감기 | 배변 (아이콘, 중앙 배치)
- 선택한 기능의 메인 이미지 표시
- 배변 실행 중: 하단에 ◀ 틸팅 | 평탄화 | 틸팅 ▶ 버튼 표시

**조작 방법**:

| 버튼 | 동작 |
|------|------|
| **환자케어** | 홈 화면으로 복귀 |
| **◀ / ▶** | 서브 기능 전환 (머리감기↔배변) |
| **확인** | 선택한 기능 실행/정지 (토글) |

**배변 모드 실행 중 추가 조작**:

| 버튼 | 동작 |
|------|------|
| **◀** (LEFT) | 좌 틸팅 명령 전송 |
| **▶** (RIGHT) | 우 틸팅 명령 전송 |
| **▼** (DOWN) | 평탄화 - 틸트 원위치 복귀 (CMD2_MOVE_CENTER, 0x44) |
| **확인** | 배변 동작 정지 (PAUSE 전송) → ◀/▶로 모드 전환 가능 |

**조작 흐름**:
```
[머리감기 선택]  ◀/▶  [배변 선택]
                        │
                    확인 누름
                        ▼
                 [배변 실행 중]
                 ◀ = 좌 틸팅
                 ▶ = 우 틸팅
                 ▼ = 평탄화
                        │
                    확인 누름
                        ▼
                  [배변 정지]
                 ◀/▶ = 모드 전환 가능
```

> **참고**: 환자케어 모드에서는 설정/저장 버튼이 비활성됩니다.

---

### 6.5 온열 (HEAT)

온열 버튼을 누르면 **화면 전환 없이** 현재 화면을 유지하면서 온열 단계만 순환합니다. 리모컨 LED로 현재 단계를 확인합니다.

**조작 방법**:

| 버튼 | 동작 |
|------|------|
| **온열** | 세기 단계 전환 (OFF → 1단 → 2단 → 3단 → OFF 순환) |

온열 버튼을 반복 누를 때마다 단계가 순환됩니다. 현재 단계에 해당하는 리모컨 LED가 누적 점등됩니다.

**온열 단계 및 LED**:
| 단계 | 명령 | LED 상태 |
|------|------|----------|
| OFF | CMD2_HEAT (0x50) | 모두 소등 |
| 1단 | CMD2_HEAT1 (0x51) | LED 1개 점등 (P3.2) |
| 2단 | CMD2_HEAT2 (0x52) | LED 2개 점등 (P3.2, P3.1) |
| 3단 | CMD2_HEAT3 (0x53) | LED 3개 점등 (P3.2, P3.1, P3.0) |

> **참고**: 별도의 온열 화면이 없으며, 어떤 화면에서든 온열 버튼으로 바로 제어 가능합니다.

---

### 6.6 통풍 (VENTILATION)

통풍 버튼을 누르면 **화면 전환 없이** 현재 화면을 유지하면서 통풍 단계만 순환합니다. 리모컨 LED로 현재 단계를 확인합니다.

**조작 방법**:

| 버튼 | 동작 |
|------|------|
| **통풍** | 세기 단계 전환 (OFF → 1단 → 2단 → 3단 → OFF 순환) |

통풍 버튼을 반복 누를 때마다 단계가 순환됩니다. 현재 단계에 해당하는 리모컨 LED가 왼쪽부터 누적 점등됩니다.

**통풍 단계 및 LED**:
| 단계 | 명령 | LED 상태 |
|------|------|----------|
| OFF | 0x60 | 모두 소등 |
| 1단 | 0x61 | LED 1개 점등 (P3.3) |
| 2단 | 0x62 | LED 2개 점등 (P3.3, P3.4) |
| 3단 | 0x63 | LED 3개 점등 (P3.3, P3.4, P4.4) |

> **참고**: 별도의 통풍 화면이 없으며, 어떤 화면에서든 통풍 버튼으로 바로 제어 가능합니다.

### 6.7 초기화 모드 (INITIAL)

초기화 버튼을 눌러 진입합니다. 모든 설정을 공장 초기값으로 되돌립니다.

**화면 구성**:
- 진한 네이비 배경 + 상단 장식 라인
- 둥근 타이틀 바: "초기화"
- 초기화 전: 주황색 경고 원형 아이콘 (!) + "시스템을 초기화하려면 / 확인 버튼을 누르세요" 안내
- 초기화 후: 초록색 성공 원형 아이콘 (OK) + "시스템 초기화 완료" 메시지

**조작 방법**:

| 버튼 | 동작 |
|------|------|
| **초기화** | 홈 화면으로 복귀 |
| **확인** | 초기화 실행 |

**초기화 범위**:
교대부양, 체압분산, 마사지, 환자케어, 온열, 통풍의 모든 설정값이 기본값으로 복원됩니다.

> **주의**: 초기화는 되돌릴 수 없습니다. 확인 버튼을 신중히 눌러주세요.

---

### 6.8 설정/저장 모드 (SET/SAVE)

홈 화면에서 설정/저장 버튼을 누르면 매뉴얼 설정 및 자가진단 모드로 진입합니다.

**서브 메뉴**:
- **매뉴얼**: 수동 설정
- **자가진단 (Self Test)**: 리모컨 LED, 오디오, 팬, 히터, 액추에이터, BAR 센서, 모터, RPM, 모터 보드 LED 등을 점검

## 7. 공통 조작 패턴

### 7.1 설정 모드 진입/해제

대부분의 모드(체압분산, 교대부양, 마사지)에서 공통적인 설정 패턴:

```
[일반 모드]  ──설정/저장──▶  [설정 모드]  ──설정/저장──▶  [저장 후 일반 모드]
```

1. **설정/저장** 1회 누름: 설정 모드 진입 (신체부위 선택 커서 활성화)
2. **▲/▼**: 신체부위 선택 (머리/몸통(상)/몸통(하)/다리)
3. **◀/▶**: 설정값 쪽으로 커서 이동 후 값 조절 (감소/증가)
4. **확인**: 선택한 부위 Enable/Disable 토글
5. **설정/저장** 다시 누름: 현재 설정값 저장 및 침대로 전송

### 7.2 시작/일시정지/재시작

설정 모드가 아닌 일반 모드에서 확인 버튼의 동작:

```
[정지]  ──확인──▶  [동작 중]  ──확인──▶  [일시정지]  ──확인──▶  [재동작]
```

### 7.3 모드 종료

각 모드에서 **같은 모드 버튼을 다시 누르면** 홈 화면으로 복귀합니다.
- 예: 교대부양 모드에서 교대부양 버튼 → 홈 화면

### 온열 LED (P3.0, P3.1, P3.2) - 누적 점등
| 단계 | P3.2 | P3.1 | P3.0 | 시각 |
|------|:----:|:----:|:----:|------|

| 1단 | ON | OFF | OFF | ● ○ ○ |
| 2단 | ON | ON | OFF | ● ● ○ |
| 3단 | ON | ON | ON | ● ● ● |

### 통풍 LED (P3.3, P3.4, P4.4) - 왼쪽부터 누적 점등
| 단계 | P3.3 | P3.4 | P4.4 | 시각 |
|------|:----:|:----:|:----:|------|
| OFF | OFF | OFF | OFF | ○ ○ ○ |
| 1단 | ON | OFF | OFF | ● ○ ○ |
| 2단 | ON | ON | OFF | ● ● ○ |
| 3단 | ON | ON | ON | ● ● ● |

## 9. ESP32 통신 프로토콜

리모컨과 침대 본체는 ESP32 모듈을 통해 UART 통신합니다.

### 패킷 구조

```
| STX (2B) | PROD_CODE (4B) | CMD_ID (1B) | ACT_CMD (1B) | LENGTH (2B) | DATA (nB) | CRC16 (2B) |
| 0xFF81   | "SB1C"         |              |               |              |            |             |
```

### 주요 명령어

| CMD_ID | 설명 | 방향 |
|--------|------|------|

## 11. 주의사항

1. **전원 OFF 상태**에서는 전원 버튼만 동작하며, 다른 모든 버튼은 무시됩니다.
2. **전원 종료 중** ("종료합니다." 화면)에는 전원 버튼 포함 모든 버튼이 차단됩니다.
3. 각 모드에서 **다른 모드 버튼을 누르면** 해당 모드로 즉시 전환됩니다.
4. **설정/저장**을 누르지 않고 다른 모드로 이동하면 변경한 설정값은 **저장되지 않습니다**.
5. **초기화**는 모든 설정을 공장 초기값으로 되돌리므로 주의하세요.
6. **온열/통풍**은 별도 화면 없이 어떤 모드에서든 버튼으로 즉시 제어됩니다.
7. **배변 모드** 실행 중 ◀/▶는 틸팅 명령으로 동작하며, **확인** 버튼으로 정지 후 모드 전환이 가능합니다.
